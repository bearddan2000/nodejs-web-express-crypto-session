# nodejs-web-express-crypto-session

## Description
A middleware sets the session
name from express default to an
unique name.

## Tech stack
- nodejs
- npm
  - express
  - express-session

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
